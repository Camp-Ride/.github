# 캠프라이드 (CampRide)

## 📱 프로젝트 소개

캠프라이드는 교통비 절약이 필요한 직장인, 예비군, 대학생에게 쉽고 빠른 택시, 카풀 커뮤니티를 제공합니다.

## 🌟 주요 기능

- 커뮤니티 기능
- 방 생성, 조회 기능
- 채팅 기능

## 🔧 기술 스택

### Frontend

- Flutter

### Backend

- SpringBoot
